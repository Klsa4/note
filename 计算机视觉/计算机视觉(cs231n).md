## 两种流行的方法和分类器
### First classifier: Nearest Neighbor
#### train: 把所有数据点保存下来
#### predict: 根据距离公式计算出距离最近的训练数据点，预测出的类别就是训练数据点的类别

#### train: 把所有数据点保存下来
#### predict:  根据距离公式计算出距离最近的k个训练数据点，再由类别最多的数据点的类型决定该数据点类型
#### 距离公式一般用曼哈顿距离，因为能保存特征

# 正则化与优化(Regularization and Optimization)
## 正则化(Regularization)定义：阻止模型在训练集上发挥得太好（防止过拟合）
### 实例：$L(w) = \frac{1}{N}\sum_{i = 1}^{N}L_{i}(f(x_i,W), y_i) + {\lambda}R(W)$
- $\lambda$是正则化力量(Regularization Strength)，且为超参数
### 为什么要正则化？
- 表示对权重的偏好
- 使得模型在测试数据集上运行效果更好
- 通过增加斜率促进优化
### 正则化通常不会使函数变复杂，这取决于具体应用
## 优化(Optimization)定义：使参数随着slope方向变化
```
在一维函数中看负斜率
在多维函数中看负梯度的方向，此方向为参数变化后精准度提升最快的方向
```
### 数值梯度(Numerical Gradient)
- 很慢，需要循环所有维度
- 估计值
- 编码很简单
### 解析梯度(Analytical Gradient)
- 准确
- 快速
- 容易出错
### 梯度下降(Gradient Descent)
#### Stochastic Gradient Descent (SGD)
- 实现：训练模型时每次循环在训练集中不重复地随机选择子训练集进行训练
- 可能存在的问题
	- 过度调整，梯度方向改变很大（主要问题）
	- 抵达局部最小值或者鞍点（梯度为零）会卡住
	- 因为每次取的都是子集进行训练，梯度下降时会有噪音，方向不明确
#### SGD + Momentum
- 定义：在SGD的基础上，把上一次的速度与现梯度相加
- 实现：
``` Python
vx = 0
while True:
	dx = compute_gradient(x)
	vx = rho * vx + dx
	x -= learning_rate * vx
	# vx变量是混合了上次迭代的梯度和这次的梯度，有利于克服鞍点、局部最小值、过度调整和方向不明确等问题
	# 其中rho变量和learning_rate为超参数，需要手动调节
	# rho变量可以简单理解成动量残留率
```
- 解决的问题
	- 梯度方向的改变被抵消
#### RMSProp
- 实现：```
``` Python
grad_squared = 0
while True:
	dx = compute_gradient(x)
	grad_squared = decay_rate * grad_squared + (1 - dacay_rate) * dx * dx
	x -= learning_rate * dx / (np.sqrt(grad_squared) + 1e-7)
	# 其中decay_rate和learning_rate为超参数，需要手动调节
	# decay_rate从字面理解就是衰减率，指代的是上次迭代中梯度平方的残留率，而(1 - decay_rate)刚好就是此次梯度平方的比率
	# 除以累积的梯度平方有利于在平坦处增大有效步长，在狭窄的山谷上缩小步长。
	# 不难看出，由于，在平坦处的梯度会变小，积累的梯度平方也会变小，相除之后
	
```
#### Adam
- 实现：(almost)
``` Python
first_moment = 0
second_moment = 0

while True:
	dx = compute_gradient(x)
	first_moment = first_moment * beta1 + (1 - beta1) * dx # 结合momentum
	second_moment = second_moment * beta2 + (1 - beta2) * dx * dx # 结合RMSProp
	x -= learning_rate * first_moment / (np.sqrt(second_moment) - 1e-7)
	
	# 将前面描述的两种算法结合在一起，既能修正方向，又能修正步长
	# 这段代码有个问题，在刚启动的时候由于beta2的值非常接近1，(1 - beta2)的值非常接近0，second_moment的值就会非常的小，导致步长非常大
```
- 实现：(full form)
``` Python
first_moment = 0
second_moment = 0
for i in range(1, num_iterations + 1):
	dx = compute_gradient(x)
	first_moment = first_moment * beta1 + (1 - beta1) * dx
	second_moment = second_moment * beta2 + (1 - beta2) * dx * dx
	first_unbias = first_moment / (1 - beta1 ** i)
	second_unbias = second_moment / (1 - beta2 ** i)
	x -= learning_rate * first_unbias / (np.sqrt(second_unbias) + 1e-7)
	# 在第一次迭代时，(1 - beta1 ** i) 和 (1 - beta2 ** i)正好抵消(1 - beta1)和(1 - beta2)，随着迭代次数增加，(1 - beta1 ** i) 和 (1 - beta2 ** i)逐渐增大，淡化作用
```
- beta1 = 0.9 和 beta2 = 0.999 和 learning_rate = 1e-3 or 5e-4 是很棒的开始训练的数据点

### 学习率
- 以上梯度函数都选择学习率作为超参数
- 在训练过程中调整学习率是很好的方案
#### Learning Rate Decays Over Time
- Step：在固定循环次数后减少学习率。例如在ResNets中，在循环30，60和90次后将学习率乘0.1
- Cosine：$\alpha_t=\frac{1}{\alpha_0}(1 + cos(t\pi/T))$
	- ![[Pasted image 20250911144949.png]]
- Linear: $\alpha_t=\alpha_0(1-t/T)$
	- ![[Pasted image 20250911145207.png]]
- Inverse Sqrt: $\alpha_t = \alpha_0 / \sqrt{t}$
	- ![[Pasted image 20250911145633.png]]
- Linear Warmup
	- 说明：高的学习率可能造成学习率爆炸；在最初的约5000次迭代中线性地增加学习率可以阻止这种情况
- 经验法则：如果你把每批训练增加至N倍，那么初始学习率也要扩大到N倍
- 注
	- $\alpha_0$: 初始学习率
	- $\alpha_t$: 在第t次循环的学习率
	- T: 总循环次数

## 关于正则化
- $\lambda$的值越大，正则化越强，对参数的限制就越强，边界越粗略
- $\lambda$要选一个合适的值，不能太大也不能太小
- 经验法则：扩大网络规模不能作为正则化手段
## 如何在神经网络计算梯度？
- (Bad) Idea: Derive $\Delta_W L$ on Paper
	- 问题1：非常冗长繁琐，需要很多纸
	- 问题2：如果我们想要临时改变损失函数呢?比如说使用Softmax代替Hing，需要从头开始计算
	- 问题3：非常复杂的模型不能够计算出来
- Better Idea: Computational graphs + Backpropagation ()
	- 反向传播的简单理解：以链式法则为原理构建出的利用上游梯度和本地梯度(需要用输入值计算出)相乘来求出下游梯度，特点是易模块化

### Dropout Layer(sometimes)
- 是一种正则化方法
- 在每个向前传播过程，随机使一些神经元的参数归零，来达到特征的冗余表示。
- 每个神经元有一定概率被丢掉参数，这个概率是超参数，一般取0.5
- 由于训练过程中会丢掉某些参数，在后续验证时要将每个参数乘以0.5

# Recurrent Neural Network
```
将一个序列转换成另一个序列
```
## 核心思想：RNNs有一个内部状态，当一个序列被处理时它会被更新
## 公式
- 新状态公式：$h_t=f_W(h_{h-1},x_t)$
- 输出公式：$y_t=f_{W_{hy}}(h_t)$
## 梯度下降：计算每个时间步的和
- 可能的问题：时间步太多导致计算机内存装不下递归栈
	- 解决方法：分块计算梯度
## RNN变种：Long Short Term Memory(LSTM)
- 为了解决RNN的过长导致梯度消失的问题
## 现代RNN优点
- 不限上下文长度
- 计算量随着序列长度增长线性增长
## 别称state space model
